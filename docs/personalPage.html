TALLER SOBRE HTML Y CSS

El diseño de estas páginas es desarrollado por Jefry Cardona utilizando componentes de Bootstrap, estilos de Google, hojas de estilo.

Elementos usados en las paginas web

A continuación se listárán algunos de los elementos a mi consideración más importantes en la creación y funcionamiento de estas paginas web.

Bootstrap

  • Carousel: es un componente basado en javascript contenido en Bootstrap que nos permite presentar contenido visual de forma interactiva en nuestra página web, en nuestro caso una serie de imagenes representativos al tema que se expone. El componente de bootstrap al que se hizo referencia es carousel slide carousel-fullscreen carousel-fade.

  • Botones de control para el carrusel: nos permite desplazarnos entre los diferenes elementos visuales que están contenidos dentro del carrusel. Las referencias que se hicieron fueron las siguientes: left carousel-control y right carousel-control.

  • Carousel indicators: permite visualizar la cantidad de elementos visuales hay contenidos dentro del carrusel y cual estamos visualizando actualmente. La referencia para utilizar este componente es carousel-indicators.

  • Carousel inner: permite vincular los elementos o slides dentro del carrusel. La referencia para utilizar este componente es carousel-inner.

  • Carousel caption: permite agregar texto a cada uno de los slides que componen el carrusel. La referencia para utilizar este componente es carousel-caption.

  • blockquote: se usa para escribir una cita textual o un párrafo exacto y que éste se diferencie del resto del texto. El efecto que nos muestra es que crea unos espacios a la derecha y a la izquierda del texto, para que se diferencie de los demás. La referencia hecha es <blockquote></blockquote>.

CSS

  • Transiciones en el carrusel: agrega efectos y modos en la visualización cuando se realizan desplazamientos entte los slides del carrusel. Se usaron los siguientes comandos: -webkit-transition-property, transition-property, -webkit-transform:, transform
